# vim-solidity
Vim syntax file for solidity

packaged for [Vundle](https://github.com/VundleVim/Vundle.vim)

To install, install Vundle, then add to ~/.vimrc

```
Plugin 'duanenielsen/vim-solidity'
```

Then open vim and type the vim command 

```
:PluginInstall!
```
